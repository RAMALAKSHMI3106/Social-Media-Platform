# Social Media Platform - Mini Project

## Description

This is a Java Mini Project developed using Java Swing in Eclipse IDE. The application simulates a basic social media platform where users can register, log in, create and view posts, and manage their profiles through a simple and user-friendly interface. This project demonstrates Java GUI development, event handling, and object-oriented programming concepts.

## Features

- User Registration
- User Login
- Create and View Posts
- Home Page
- Modify Profile

## Technologies Used

- Java
- Java Swing
- Eclipse IDE
- Object-Oriented Programming (OOP)
